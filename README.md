# Frontend AI Internship

My learning repo for the Frontend AI Engineering Internship. I'm building React applications with AI-assisted development while practicing modern frontend patterns, component design, and Git workflows.

## Tech Stack

- React 19
- Vite
- TypeScript
- ESLint
- Tailwind CSS (planned)

## Prerequisites

- Node.js 18+
- npm 9+

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the development server |
| `npm run build` | Type-check and create a production build |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |

## Project Structure

```
frontend-track/
├── public/          # Static assets
├── src/

│   ├── App.tsx      # Root component
│   └── main.tsx     # Application entry point
├── eslint.config.js

```
